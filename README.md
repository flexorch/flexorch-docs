# FlexOrch Documentation

Source for [doc.flexorch.com](https://doc.flexorch.com) — built with [Mintlify](https://mintlify.com).

## Development

Install the Mintlify CLI:

```bash
npm install -g mintlify
```

Run locally:

```bash
mintlify dev
```

## Contributing

Edit `.mdx` files. The `mint.json` controls navigation — update it when adding new pages.

The `/api-reference` section is auto-generated from the [OpenAPI spec](https://api.flexorch.com/openapi.json). No manual edits needed there.
