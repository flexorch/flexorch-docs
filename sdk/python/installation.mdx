---
title: Installation
description: Install the FlexOrch Python SDK.
---

```bash
pip install flexorch-sdk
```

Requires Python 3.10+.

```python
from flexorch_sdk import FlexOrch

client = FlexOrch(api_key="dfx_your_key_here")
```

Set your API key as an environment variable to avoid hardcoding it:

```bash
export FLEXORCH_API_KEY=dfx_your_key_here
```

```python
import os
from flexorch_sdk import FlexOrch

client = FlexOrch(api_key=os.environ["FLEXORCH_API_KEY"])
```

See [Quickstart](/sdk/python/quickstart) for your first document processing example.
