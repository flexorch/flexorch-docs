## Prerequisites

- Python 3.10+
- A FlexOrch account and API key — generate one under **Settings → API Keys**

---

## Install

```bash
pip install flexorch-sdk
```

---

## Full example

```python
from flexorch_sdk import FlexOrch

client = FlexOrch()  # reads FLEXORCH_API_KEY from env

# 1 — Upload and process a document
job = client.jobs.upload("invoice.pdf")
print(job.id)  # "job_abc123"

# 2 — Wait for the pipeline to finish
done = job.wait_until_done(timeout=120)
print(done.status)         # "completed"
print(done.quality_grade)  # "A"

# 3 — Build a dataset from this job
dataset = client.datasets.build(name="q1-invoices", job_ids=[done.id])

# 4 — Export as JSONL for fine-tuning
client.datasets.export(dataset.id, format="jsonl", path="output.jsonl")

print("Done — output.jsonl written")
```

---

## Error handling

```python
from flexorch_sdk import (
    FlexOrch,
    JobFailedError,
    JobTimeoutError,
    QuotaError,
)

client = FlexOrch()

try:
    job = client.jobs.upload("document.pdf")
    done = job.wait_until_done(timeout=120)
    print(done.quality_grade)
except JobFailedError as e:
    print("Processing failed:", e.reason)
except JobTimeoutError as e:
    print("Timed out waiting for job:", e.job_id)
except QuotaError:
    print("Credit quota exceeded — upgrade your plan")
```
